### 万字文档+科创微应用小程序+Uniapp+SSM+Vue
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL、Tomcat7.5-9、微信开发者工具/HBuilderX
技术栈：Java、MySQL、Tomcat7.5-9、Spring、SpringMVC、Mybatis/MybatisPlus、Vue2、Uniapp、ElementUI、HTML、JavaScript、CSS、图片上传

系统功能模块：
本系统包含管理员、用户、企业3个角色。
管理员角色：
- 个人中心：管理个人信息，包括账户设置等。
- 用户管理：对系统中的用户账号进行增删改查操作。
- 企业管理：对企业信息进行维护，包括企业的注册、审核、信息更新等。
- 场地信息管理：管理平台上的场地基本信息，如地址、容量、设施等。
- 场地类型管理：定义和维护场地的不同类型，便于分类和检索。
- 预约参观管理：处理用户的参观预约请求，审批和安排参观事项。
- 场地预约管理：管理用户对场地的预约申请，包括审批、调度和冲突处理。
- 活动信息管理：发布、编辑和删除平台上的活动信息，保证信息的时效性与准确性。
- 报名信息管理：处理用户参与活动的报名信息，统计报名情况。
- 我的收藏管理：维护用户的收藏夹，记录用户感兴趣的场地或活动。
- 试题管理：设计和维护考试题库，用于在线测试或资格审核。
- 试卷管理：编排试题生成试卷，设置考试规则。
- 系统管理：进行系统配置、权限分配等后台管理操作。
- 考试管理：组织在线考试，包括发布考试、监控考试进程、评估成绩等。
用户角色：
- 注册登录：用户可以注册账户并登录系统。
- 预约信息：浏览并预约场地或活动。
- 参加考试：参与系统提供的在线考试。
- 个人收藏：收藏感兴趣的场地或活动信息以便后续查看。
企业角色：
- 注册登录：企业用户注册并登录系统。
- 发布信息：发布企业相关活动、招聘信息等。
- 场地管理：如果是提供场地的企业，还可以管理自己发布的场地信息。
